Planning our project

    HTML
  1. Our aim is to create a wep app that finds all of the crimes in your local area that have happened in the month of your choice. We will use open data by inserting into the API URL: the latitude, the longitude and the date.
  2. Our project will help people by showing them the crimes in a specific geographic area, and this will help them know how safe this area is and what the different crimes are. This will mean the public are better informed and it enables them to take more responsibility for themselves and belongings.
  3. There are a number of different approuches to creating this:
    • Make an app using javascript and C++
    • Make a piece of hardware so that when people press a phisical button, the info is displayed on a seven-digit display (sliding across the screen so it fits.)
    • Make a graphical interface for this (as a program)
    • Make a web app using a combination of HTML, CSS and JavaScript

We chose this last approach as we think this is the most user freindly way of accessing infomation.

This approach also uses the existing skills we have as a team, but also the chance to develop further experience and skills with the help of our mentors.
